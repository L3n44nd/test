Обработчик файлов 

Сборка:
- Открыть CMakeLists.txt в Qt Creator
- Выбрать комплект MinGW
- Нажать "Собрать"

Функциональность : 
- модификация файлов XOR с 8-байтным ключом
- чтение больших файлов буферами (10 МБ)
- отдельный поток для обработчика
- шкала прогресса
- возможность мгновенной остановки без повреждения файлов
- валидация полей 

Cтруктура: fileModifier - главное окно, fileProcessor - обработчик
